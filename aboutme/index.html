Projects

Below are some of my hobby projects that I have been working on in my free time.

Numodrillo

As I started to leverage my Italian during traveling I soon realized how slowly I am able to comprehend numbers, even though I know the rules. So I thought I need serious number drills and since I did not find existing tools, I created Numodrillo (pun intended).

The current version works for 4 languages and 3 difficulties (number ranges). I have several feature ideas that I might work on in the future.

Technologies:

  • native Android app written in Kotlin and Jetpack Compose
  • used ttsmaker.com for text to speech
  • supporting config scripts written in Python
  • leveraged Gemini during development

Memcard Quiz

I created this game from scratch in an attempt to combine the classical memory card game with picture based quiz questions. The goal is to exercise short term memory and challenge the player ro recognize and name various objects in their native or learnt language.

The current version has several built in decks of memory cards and ability to track high scores locally. Several feature ideas are waiting patiently in the backlog.

Technologies:

  • native Android app written in java
  • built using the LibGDX framework
  • supporting config scripts written in Python
  • created artwork from scratch using Inkscape
  • pictures are free to use commercially

runstat

Being a runner with an analytical mind I have always wanted to learn more about my runs -- more than what popular platforms such as Strava provide. Hence, developed a set of scripts in an attempt to answer some of the questions.

All scripts were written in Python from scratch.

  • plotted all my runs on an interactive map
  • generated a scatter that visualizes the best 5 times on each round km distance
  • generated a table of 5 best times on each round km distance

svorg

There might be a bunch of similar tools available, but I thought it would be fun to roll my own: svorg is a simplistic CLI tool to generate static SVG org chart from JSON data and templates.

The script was written in Python from scratch.

Source is available on GitHub at https://github.com/sabbiasoftware/svorg.
